# Interactive Data Analytics Dashboard

# Overview
This project is an interactive Power BI dashboard designed to analyze and visualize data for better decision-making.

# Objectives
- Transform raw data into meaningful insights
- Visualize KPIs and trends interactively
- Support data-driven decisions

# Tools & Technologies
- Power BI
- Microsoft Excel / CSV
- Data Visualization

# Key Features
- Interactive charts and slicers
- KPI indicators
- Clean and intuitive layout
